== Description ==
Automatically generate PDF invoices and attach them to every form submission in Gravity Forms. This is the ultimate self-hosted solution for dynamically creating PDF invoices. It's easy to set up in just 3 minutes. Simply add your billing details in the settings and configure the form fields in Gravity Forms feeds. The plugin will automatically generate invoices from the pricing fields in Gravity Forms.
